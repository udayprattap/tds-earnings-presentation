Data Analytics Pipeline

Python code for revenue trend analysis:


import pandas as pd
import numpy as np

# Load quarterly revenue data
df = pd.read_csv('revenue_data.csv')
df['growth_rate'] = df['revenue'].pct_change() * 100
df['moving_avg'] = df['revenue'].rolling(window=4).mean()

# Calculate YoY metrics
yoy_growth = ((df.iloc[-1]['revenue'] - df.iloc[-5]['revenue']) 
               / df.iloc[-5]['revenue']) * 100
print(f"YoY Growth: {yoy_growth:.2f}%")

Database Query Optimization

Customer segmentation query:


SELECT 
    customer_segment,
    COUNT(*) as total_customers,
    AVG(account_balance) as avg_balance,
    SUM(transaction_volume) as total_volume,
    MAX(last_transaction_date) as recent_activity
FROM customers
WHERE registration_date >= '2025-07-01'
GROUP BY customer_segment
ORDER BY total_volume DESC;

Portfolio Risk Calculation

Value at Risk (VaR) - 95% confidence level:

$$ VaR_{95} = \mu - 1.645 \times \sigma $$

Where: $\mu$ = expected return, $\sigma$ = standard deviation

Our Portfolio VaR: $2.3M (within acceptable limits)

Sharpe Ratio for risk-adjusted returns:

$$ S = \frac{R_p - R_f}{\sigma_p} $$

$R_p$ = portfolio return, $R_f$ = risk-free rate, $\sigma_p$ = portfolio std dev

Our Sharpe Ratio: 1.85 (excellent risk-adjusted performance)
